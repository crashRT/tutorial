# 1.2. Blender を使ってみる

## 概要
この節では、Blender の画面の見方について解説した後、視点操作、移動・回転・拡大縮小、及びレンダリングについて扱います。

## Blender の起動画面

Blender を起動すると以下のような画面が表示されると思います。

起動時には、Topbar, Status Bar の他に以下の「エリア」と呼ばれるものが表示されています。
- 3D Viewport
- Outliner
- Properties
- Timeline

![起動画面の説明](./img/1.2.0.png)
